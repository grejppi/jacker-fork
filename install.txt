Quick Build Instructions (Ubuntu Linux)
=======================================


Prerequisites
-------------

 * gtkmm-2.4
 * sigc++-2.0
 * libjack
 * SCons

Build
-----

 1. From the source root directory, 
    enter into a terminal:
 
    $ scons
    
 2. The project will build.
 
Install
-------

 1. To install jacker in your system, use
 
    $ sudo scons install

Run
---

 1. After installing, just run:
    
    $ jacker
